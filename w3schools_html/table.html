테이블 선 css

  • dotted : 점선
  • dashed : 파선
  • solid : 실선
  • double : 이중선
  • groove : 입체적으로 들어간 느낌을 주는 선
  • ridge : 입체적으로 튀어나온 느낌을 주는 선
  • inset : 안으로 들어간 듯한 테두리
  • outset : 밖으로 나온듯한 테두리
  • none : 테두리 없음.
  • hidden : 테두리 숨기기

테이블 폭 css

  • width : 테이블 열 너비
  • height : 테이블 행 높이

테이블 셀 패딩

  • padding : 셀 내부의 패딩과 셀 사이의 공간조정.
  • spacing : 셀 간격

테이블 헤더

  • th : 테이블의 헤더
  • caption : 표 캡션(제목역할을 하는 캡션)

테이블 합침

  • colspan : 셀 열 합침
  • rowspan : 셀 행 합침

테이블 스타일링

  • nth-child(even) : 1,3,5열 등에서 스타일이 이루어짐
  • nth-child(odd) : 2,4,6열 등에서 스타일이 이루어짐
  • rgba : 색상의 투명도를 지정하기 위한 색상

테이블 콜그룹

  • colgroup : 테이블의 컬럼에 적용할 스타일을 해당태그에서 미리 적용할수 있게 도와줌.
